Vaibot is always intended to make things easier. In here we have taken the idea of chatbot to a new level. In Vaibot Version 1.1 you can contribute to vaibot just by adding text files!!! So  Lets see hoe it Works

Instractions for adding conversations to vaibot.
1. Open a new text file in .txt format.
2. Just write The question you want to ask to Vaibot in the first line and your desired answere in following line.
3. Vaibot will automatically search all present strings to produce answere strings 
4. See exampleexample.txt for examples.
